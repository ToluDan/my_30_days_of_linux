## What I Learned

The ps command in Linux is used to display information about currently running processes. It provides a snapshot of processes at the time the command is executed. System administrators commonly use it to monitor processes, troubleshoot issues, and identify resource usage. ps shows static process information rather than continuously updating output.

Display currently running processes on a Linux system
Identify process IDs (PIDs) for management tasks like stopping processes
View process ownership, CPU usage, and memory consumption
Inspect background processes and system services
Filter processes based on users, IDs, terminals, or command names
when you type ps the following displays
PID: the unique process ID
TTY: terminal type that the user is logged into
TIME: amount of CPU in minutes and seconds that the process has been running
CMD: name of the command that launched the process

some PS optiosn include 
-e : Display All Running Processes
-f : Display Full Process Format
-u : Display Processes for a Specific User
-x : Show Processes Without a Controlling Terminal
-a : Display Processes for All Users With a Terminal
 -A : View All Running Processes in Linux
-N or --deselect : Exclude Specific Processes
